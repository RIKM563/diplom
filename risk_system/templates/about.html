О системе

Веб-приложение предназначено для автоматизации анализа и оценки рисков информационной безопасности корпоративных систем банка на основе методов машинного обучения.

Основные возможности

  • обучение модели вероятности реализации угроз;
  • калибровка вероятностных оценок;
  • расчет ущерба, критичности и итогового риска;
  • учет зависимостей угроз и графового распространения риска;
  • классификация риска по уровням;
  • подбор мер защиты при ресурсных ограничениях.

Логика работы

1. Обучение Система принимает обучающие данные и строит модель оценки вероятности угроз.
2. Оценка Для новых входных данных вычисляются вероятности, ущерб, критичность и итоговые классы риска.
3. Оптимизация По рассчитанным рискам подбираются меры защиты с учетом ограничений.

Формат объяснения результатов

В интерфейсе результаты показываются в трех формах:

  • сводные карточки для быстрого восприятия общей картины;
  • таблицы по событиям, узлам, активам и выбранным мерам защиты;
  • интерпретационные блоки, где для каждого узла кратко описываются класс риска, уровень риска и основные влияющие признаки.
